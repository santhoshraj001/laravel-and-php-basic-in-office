Condition	What toggle does

If element already HAS "selected"	➝ Remove the class

So one click adds it, second click removes it.

EXAMPLE PROGRAM AND METHODS EXPLANATION
function selectTemplate(selected) {
    document.querySelectorAll(".hover").forEach(function(item) {
        item.classList.remove("selected");
    });

    selected.classList.add("selected");
}
Method	               What it means
querySelectorAll()  - Select all matching elementsIt finds all
                     elements that match a CSS selector (like .hover, #id, div, etc.)

forEach()	        - Loop through each selected element,
                     It loops through all items (like a list) one by one.

classList.remove()  - Remove a CSS class,
classList.add()     - Add a CSS class,

----database basic concepts in laravel----
1).env File

Used for project settings
Stores database details   (eg = DB_DATABASE=laranstd)
Laravel reads this first
remove # from default
2)Migration

Migration = DB structure file
Written in PHP
Used to create tables
Create migration
(cmd = php artisan make:migration create_larastudent_table)
up()    → create table
down()  → delete table
---if you already created a table no need to create
another table in migration file simply skip that file---
3)Common Migration Columns
$table->id();              // primary key
$table->string('name');    // text
$table->integer('age');    // number
$table->timestamps();      // created_at, updated_at

4)Run Migration
cmd = php artisan migrate

Creates table in database
View table in phpMyAdmin
